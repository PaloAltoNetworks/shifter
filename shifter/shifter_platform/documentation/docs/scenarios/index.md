# Scenarios

A scenario defines what your range contains. Choose based on your demo needs.

## Available Scenarios

| Scenario | Instances | NGFW | Best For |
|----------|-----------|------|----------|
| [Basic Range](basic-range.md) | Kali, Workstation | No | Quick demos, simple attacks |
| [AD Attack Lab](ad-attack-lab.md) | Kali, DC, Workstation | No | Active Directory attacks |
| [Basic Range with NGFW](ngfw-range.md) | Kali, Workstation | Yes | Traffic logging demos |
| [Cortex BYOT](cortex-byot.md) | Kali, DC, 2 Workstations, Server | Yes | Full enterprise scenarios |

## Quick Comparison

**No NGFW required:**
- [Basic Range](basic-range.md) - Fastest, simplest
- [AD Attack Lab](ad-attack-lab.md) - AD attacks without network logging

**NGFW required:**
- [Basic Range with NGFW](ngfw-range.md) - Simple + network visibility
- [Cortex BYOT](cortex-byot.md) - Full enterprise demo
