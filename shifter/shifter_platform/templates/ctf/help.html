{% trans "FAQ" %}
{% trans "Can I use hints?" %}

{% blocktrans trimmed %} Yes, but hints have a point penalty. The penalty is shown before you reveal the hint, so you can decide if it's worth it. {% endblocktrans %}

{% trans "What if I run out of attempts?" %}

{% blocktrans trimmed %} If a challenge has a maximum number of attempts and you've used them all, you won't be able to submit more flags for that challenge. {% endblocktrans %}

{% trans "How do teams work?" %}

{% blocktrans trimmed %} If the event uses team mode, you can join a team using an invite code from your team captain. Team scores are the sum of all member scores. {% endblocktrans %}

{% trans "What is a range?" %}

{% blocktrans trimmed %} A range is a dedicated lab environment provisioned for you during the event. It contains the systems you need to solve challenges. Access your range from the {% endblocktrans %} {% trans "Range" %} {% trans "page in the sidebar." %}
