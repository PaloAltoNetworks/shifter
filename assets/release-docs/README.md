# Shifter Release Documentation Package

This folder contains release-day documentation for Shifter's Cortex Catalyst launch (December 18, 2025).

## Purpose

These one-page handouts help different audiences understand:
- **Why care**: What Shifter does and why it matters to them
- **Business value**: How Shifter supports PANW conversations and drives PG growth
- **How to use**: Practical guidance for leveraging Shifter

## Demo Requirements

**Important**: Only technical sellers with XDR agents should run demos initially.

### Prerequisites for Demo Execution
- Access to customer's XDR/XSIAM tenant (or demo tenant)
- XDR agent installer uploaded to Shifter
- Technical understanding of XDR/XSIAM detection capabilities
- Familiarity with AI-driven attack scenarios

### Who Can Demo
- **Technical Sellers** (SEs/SCs with XDR access): Full demo execution
- **Account Execs**: Can observe and present value, but need SE support for live demos
- **Sales Leaders**: Strategic positioning, should coordinate with technical teams

## Release Day Strategy

**Flash in the pan approach**: All materials ready to go live immediately at launch.

1. **Pre-release** (December 17, 2025)
   - Distribute handouts to all teams
   - Brief sales leaders on positioning
   - Technical sellers review demo scenarios

2. **Launch Day** (December 18, 2025)
   - Activate portal access
   - Sales teams begin customer outreach
   - Technical sellers start booking demos

3. **First Week**
   - Collect feedback from early demos
   - Refine messaging based on customer reactions
   - Share early wins across teams

A self-service cyber range platform that lets you demonstrate AI-driven attacks against real XDR/XSIAM-protected infrastructure—all from a browser.
