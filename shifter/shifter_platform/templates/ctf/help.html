CTF Help

Getting Started
  1. Check your email for an invitation link from your event organizer.
  2. Click the link to register and set up your account.
  3. Once the event starts, navigate to Challenges to see available tasks.
  4. Solve challenges and submit flags to earn points.
  5. Check the Scoreboard to see how you rank against other participants.
How to Submit Flags

Each challenge has a hidden flag that you need to find. Flags are typically formatted as:

FLAG{example_flag_value}

To submit a flag:

  1. Navigate to the challenge detail page.
  2. Enter the flag exactly as you found it (flags are case-sensitive).
  3. Click Submit.

Some challenges may have a maximum number of attempts. Check the challenge description for details.

Scoring
  • Each challenge is worth a fixed number of points based on its difficulty.
  • Points are awarded on the first correct submission only.
  • Using a hint deducts a penalty from the points awarded for that challenge.
  • Rankings are determined by total score. In case of a tie, the participant who achieved their score first wins.
FAQ
Can I use hints?

Yes, but hints have a point penalty. The penalty is shown before you reveal the hint, so you can decide if it's worth it.

What if I run out of attempts?

If a challenge has a maximum number of attempts and you've used them all, you won't be able to submit more flags for that challenge.

How do teams work?

If the event uses team mode, you can join a team using an invite code from your team captain. Team scores are the sum of all member scores.

What is a range?

A range is a dedicated lab environment provisioned for you during the event. It contains the systems you need to solve challenges. Access your range from the Range page in the sidebar.
