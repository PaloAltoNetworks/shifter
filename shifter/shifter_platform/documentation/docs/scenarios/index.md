# Scenarios

A scenario defines what your range contains. Choose based on your demo needs.

## Available Scenarios

| Scenario | Instances | NGFW | Best For |
|----------|-----------|------|----------|
| [Basic Range](basic-range) | Kali, Workstation | No | Quick demos, simple attacks |
| [AD Attack Lab](ad-attack-lab) | Kali, DC, Workstation | Yes | Active Directory attacks |
| [Basic Range with NGFW](ngfw-range) | Kali, Workstation | Yes | Traffic logging demos |
| [Cortex BYOT](cortex-byot) | Kali, DC, Cortex Host, 2 Workstations, Server | Yes | Full enterprise scenarios |
| Cortex Deployment Experience | Kali, DC, Cortex Host, 2 Workstations, Server | Yes | Agent deployment exercises |
| Agentic Workshop | Kali + 5 challenge boxes | No | CTF-style workshops |

## Quick Comparison

**No NGFW required:**
- [Basic Range](basic-range) - Fastest, simplest
- Agentic Workshop - CTF with 6 boxes

**NGFW required:**
- [AD Attack Lab](ad-attack-lab) - AD attacks with network logging
- [Basic Range with NGFW](ngfw-range) - Simple + network visibility
- [Cortex BYOT](cortex-byot) - Full enterprise demo
- Cortex Deployment Experience - Manual agent deployment exercise
