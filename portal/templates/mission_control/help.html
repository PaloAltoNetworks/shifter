Help

Learn how to use Shifter

Quick Start
  1. Upload an agent - Go to Agents and upload your XDR or XSIAM agent installer
  2. Launch a range - Return to Dashboard, select your agent, and click Launch Range
  3. Wait for provisioning - Your range will be ready in a few minutes
  4. Open your workspace - Click the link to access your control environment
  5. Run scenarios - Use AI to configure vulnerabilities and run attacks
Managing Agents

Agents are XDR or XSIAM installers that get deployed to your victim machines.

  • Download your agent installer from your XDR/XSIAM console
  • Supported formats: .msi (Windows), .sh (Linux), .deb, .rpm
  • Give each agent a memorable name (e.g., "Acme Corp XSIAM")
  • You can upload multiple agents for different customers
Working with Ranges

A range is your isolated demo environment with a victim machine and attack tools.

  • Active - Range is running and ready to use
  • Paused - Range is stopped to save costs, can be resumed
  • Provisioning - Range is being set up, wait a few minutes
  • Destroyed - Range has been terminated

Tip: Pause your range when taking breaks to save costs. Destroy it when completely done.

FAQ

How long does provisioning take?

Usually 2-5 minutes depending on the agent type.

Can I have multiple ranges at once?

Currently limited to one active range at a time.

What happens if I forget to destroy my range?

Ranges auto-destroy after 8 hours of inactivity to prevent cost overruns.

Where do alerts appear?

In your XDR/XSIAM console - the same one you downloaded the agent from.
